# Привет, это проэкт Slovs

## Description

This application is designed to memorize English words

## Usage

1. Add file to your working directory (name_file.json):
```
{
	"in case of": "в случае",
	"public": "публичный",
	"only": "только"
}
```

2. To run the program, enter ./main.py -f name_file.json into the console.
